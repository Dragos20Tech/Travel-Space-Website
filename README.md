# Space Travel Website

Thanks to [Kevin Powell's course on Scrimba](https://scrimba.com/learn/spacetravel), I was able to create a website dedicated to space travel with confidence.  I am excited to continue building my skills in web development! This project not only improved my skills in manipulating DOM elements, but also gave me valuable insight into the effectiveness of using CSS Grid, especially in situations like creating the body layout.
